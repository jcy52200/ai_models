# 电商项目 UI设计规范

## 设计概述

素居采用**优雅简约**的设计风格，融合现代美学与温暖质感，呈现出高端家居品牌应有的品味与格调。

### 设计理念
- **克制**: 减少视觉噪音，让产品成为焦点
- **精致**: 每一个细节都经过精心打磨
- **温暖**: 中性色调营造家的温馨感
- **专业**: 清晰的层级与流畅的交互

## 字体系统

- **标题字体**: `Cormorant Garamond` - 优雅衬线体
- **正文字体**: `Lato` - 现代无衬线体

### 字号规范

| 层级 | 字体 | 大小 | 字重 | 行高 | 用途 |
|------|------|------|------|------|------|
| H1 | Cormorant | 44px | 400 | 1.1 | 页面主标题 |
| H2 | Cormorant | 34px | 400 | 1.15 | 区块标题 |
| H3 | Cormorant | 24px | 400 | 1.2 | 卡片标题 |

## 交互规范

### 动画时间

| 类型 | 时长 | 缓动函数 |
|------|------|----------|
| 微交互 | 150ms | ease-out |
| 悬停 | 300ms | ease-out |
| 展现 | 500ms | cubic-bezier(0.16, 1, 0.3, 1) |
| 页面切换 | 400ms | ease-in-out |
| 弹窗 | 300ms | cubic-bezier(0.175, 0.885, 0.32, 1.275) |

### 缓动函数
```css
--ease-expo-out: cubic-bezier(0.16, 1, 0.3, 1);
--ease-spring: cubic-bezier(0.175, 0.885, 0.32, 1.275);
--ease-smooth: cubic-bezier(0.4, 0, 0.2, 1);
```

### 悬停效果

#### 图片悬停
```css
/* 商品图片 */
transform: scale(1.05);
transition: transform 500ms var(--ease-expo-out);

/* 分类卡片 */
transform: scale(1.08);
overlay opacity: 0.8;
```

#### 按钮悬停
```css
/* 主按钮 */
background: #333333;

/* 次按钮 */
background: #000000;
color: #FFFFFF;

/* 文字按钮 */
underline width: 100%;
arrow translateX: 4px;
```

#### 卡片悬停
```css
transform: translateY(-12px);
box-shadow: 0 25px 50px rgba(0,0,0,0.12);
```

### 滚动效果

#### 视差滚动
```
背景移动速度: 0.3x
内容移动速度: 1x
```

#### 滚动展现
```
触发点: 元素进入视口 20%
动画: opacity 0→1, translateY 30px→0
时长: 600ms
缓动: expo-out
错开: 80-150ms
```

## 无障碍设计

### 色彩对比
- 文字与背景对比度 ≥ 4.5:1
- 大文字对比度 ≥ 3:1

### 焦点状态
```
outline: 2px solid #000000;
outline-offset: 2px;
```

### 减少动画
```css
@media (prefers-reduced-motion: reduce) {
  * {
    animation-duration: 0.01ms !important;
    transition-duration: 0.01ms !important;
  }
}
```
